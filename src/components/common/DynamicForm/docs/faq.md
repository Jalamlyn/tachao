## 常见问题

1. **如何处理复杂的表单逻辑？**
   - 使用 dependencies 配置处理字段依赖
   - 使用自定义渲染函数处理特殊UI
   - 合理拆分表单步骤

2. **如何实现条件字段显示？**
   - 使用 showWhen 配置
   - 在 render 函数中添加条件判断
   - 使用依赖关系控制显示逻辑

3. **如何处理文件上传？**
   - 使用 file 类型的字段
   - 在 render 函数中自定义上传组件
   - 处理好文件验证和限制

4. **如何优化打印效果？**
   - 使用 CSS 媒体查询
   - 调整打印样式
   - 控制分页位置

5. **如何处理表单数据的保存和恢复？**
   - 使用 form.reset 重置表单
   - 使用 form.setValue 设置字段值
   - 处理好数据格式转换
