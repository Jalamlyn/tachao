# 应用管理功能总结

## 1. 功能概述

应用管理模块是一个综合性的应用管理平台，允许用户创建、配置和管理多个应用实例。每个应用可以包含多个表单模板和数据报表，为用户提供一站式的应用管理解决方案。

## 2. 核心功能

### 2.1 应用管理

- **应用创建**：用户可以创建新的应用实例，只需提供应用名称即可快速创建
- **应用列表**：以卡片形式展示所有应用，包含应用状态、创建时间等关键信息
- **应用删除**：支持删除不需要的应用，删除前会进行二次确认

### 2.2 应用配置

- **表单模板配置**：可为应用关联多个表单模板，用于数据收集
- **报表配置**：可为应用关联多个数据报表，用于数据展示和分析
- **状态管理**：显示应用的运行状态（运行中/已停用）

### 2.3 应用访问

- **快速访问**：提供应用访问链接，可在新窗口打开应用
- **应用预览**：可直接查看应用内的表单和报表内容
- **权限控制**：基于用户权限控制应用的访问和管理

## 3. 用户界面

### 3.1 应用卡片展示
- 应用名称和状态标识
- 创建时间信息
- 关联的表单和报表数量统计
- 快捷操作按钮（访问、配置、删除）

### 3.2 应用配置界面
- 表单模板选择列表
- 报表选择列表
- 配置保存功能

### 3.3 应用访问界面
- 应用主页展示
- 表单填写入口
- 报表查看入口

## 4. 使用流程

1. **创建应用**
   - 点击"创建应用"按钮
   - 输入应用名称
   - 确认创建

2. **配置应用**
   - 选择需要配置的应用
   - 选择关联的表单模板
   - 选择关联的数据报表
   - 保存配置

3. **使用应用**
   - 通过应用卡片的"访问应用"按钮进入应用
   - 选择需要填写的表单或查看的报表
   - 进行相应操作
